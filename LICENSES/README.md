# License bundle

This directory collects the license texts and notices needed for Verdad source
and binary distributions.

Verdad-authored code:

- `GPL-2.0-only.txt`
- `Unlicense.txt`

Direct third-party components used by Verdad:

- `FLTK-LICENSE.txt`: FLTK license text, including the FLTK static-linking
  exception and the LGPL text it ships with.
- `SWORD-LICENSE.txt`: CrossWire SWORD license text from the local SWORD source
  tree.
- `litehtml-BSD-3-Clause.txt`: litehtml license text.
- `gumbo-Apache-2.0.txt`: gumbo parser license text.
- `SQLite-Public-Domain.txt`: SQLite public-domain notice.

Practical note:

- If you distribute Verdad builds that link against SWORD, comply with the
  GPL-2.0 terms for that distribution.
- Verdad is based in part on the work of the FLTK project
  (https://www.fltk.org).
