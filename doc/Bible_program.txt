Verdad current program summary

User-facing overview
- Desktop Bible study application for CrossWire SWORD modules.
- Left pane: always-visible search box, module browser, search results, tags, and shared preview area.
- Center pane: Bible reader with book/chapter/module controls, typed reference entry, verse selection, paragraph mode, and parallel Bible columns.
- Right pane: commentary, general books, and a global Documents tab across the top, with the dictionary or lexicon pane docked below.

Implemented features
- Multiple study tabs with restore-on-startup behavior.
- Search modes for multi-word, exact phrase, regex, and Strong's or lemma queries.
- Background SQLite FTS5 indexing for Bible modules, with progress feedback in the Search tab.
- Search result behavior:
  - Single-click updates the preview pane.
  - Double-click navigates the active study tab.
  - Middle-click opens the verse in a new study tab.
- Bible display tools:
  - Verse-by-verse display or paragraph mode.
  - Up to seven parallel Bible columns.
  - Toggleable visible markers for Strong's, morphology, footnotes, and cross references.
  - Word hover preview plus right-click actions for search, Strong's search, dictionary lookup, copy, and tagging.
- Commentary and reference tools:
  - Commentary follows the selected verse.
  - Commentary verse gutter links sync the Bible pane.
  - Linked verses and notes from commentary or general books can update the preview pane or populate the Search tab.
- General books:
  - Module table of contents chooser.
  - Loads a bounded page from the selected table of contents entry; very small sections can include a few following entries.
  - Inline contents links scroll inside the current rendered section.
- Documents and editing:
  - Lightweight HTML document editor with bold, italic, lists, text sizing, and horizontal rules.
  - Verse references in saved documents become clickable `sword://` links.
  - Writable commentary modules can be edited in place from the Commentary tab.
- Tags:
  - Verse-based tags with create, rename, delete, filter, and remove-verse actions.
  - Tagged verses show a single marker in the Bible pane.
  - Tags are stored separately from the search index.
- Module and app management:
  - Built-in Module Manager for local and remote SWORD sources.
  - Settings dialog for UI font, text font and size, hover delay, default Strong's dictionaries, and language-specific word dictionaries.

Persistent local files
- `~/.config/verdad/preferences.conf`
  - Window geometry, splitters, study tabs, restored session state, appearance, and dictionary preferences.
- `~/.config/verdad/module_index.db`
  - Search index database for Bible modules only.
- `~/.config/verdad/tags.db`
  - Verse tags in a separate SQLite database.

Technical summary
- Language and build system: C++17 with CMake.
- UI toolkit: FLTK.
- HTML renderer: vendored `litehtml`.
- Module access and XHTML generation: CrossWire SWORD.
- Runtime assets:
  - `data/master.css` for rendered Bible/commentary/dictionary/general-book styling.
  - `data/help.html` for the in-app help window.
- Major source areas:
  - `src/app/` application startup, preferences, session restore.
  - `src/ui/` panes, dialogs, widgets, and editors.
  - `src/sword/` module access, link resolution, HTML generation, markup cleanup.
  - `src/search/` indexing and search.
  - `src/tags/` tag management and persistence.

Current limits
- No dedicated automated test suite yet; validation is build plus manual UI checks.
- Search indexing currently targets Bible modules.
- Commentary editing only works for writable SWORD modules.
